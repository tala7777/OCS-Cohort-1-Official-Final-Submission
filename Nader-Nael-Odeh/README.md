# Legal Q&A (Ask Lawyer)

## Project Description

A comprehensive platform connecting users with verified legal professionals. Users can ask questions, browse advice, and read legal articles, while lawyers can manage their profiles and provide expert answers in a structured, professional environment.

## Features

- **Public Q&A Feed**: Reddit-style legal question threads with category filtering and sorting.
- **Verified Lawyer Profiles**: Advanced verification system for lawyers with CV uploads and admin approval.
- **Admin Dashboard**: Real-time analytics, user management, and lawyer request verification flow.
- **Dynamic Content**: Category-based management for questions, articles, and specializations.
- **Premium UI/UX**: Professional dark theme with glassmorphism effects and responsive design.

## Technologies & Tools Used

- **Backend**: Laravel, Livewire (PHP)
- **Frontend**: HTML5, Blade, CSS3 (Vanilla + Bootstrap utility), JavaScript (Vanilla)
- **Database**: MySQL
- **Tools**: GitHub, SweetAlert2, Chart.js
